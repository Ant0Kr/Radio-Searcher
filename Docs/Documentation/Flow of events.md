# Поток событий
### 1. Поиск радиодеталей
**Основной поток событий:**
1. Вариант использования начинается, когда пользователь открыл главную страницу сайта.
2. Пользователь выбирает сайты на которых будет производиться поиск, установкой флагов в чекбоксах.
3. Пользователь нажимает кнопку "Search".
4. Если пользователь не выбрал ни одного чекбокса, будет отображена ошибка, и просьба выбрать чекбокс. 
Переход к п.6. Иначе переход к п.5
5. На экране монитора отображается результат поиска.
6. Вариант использования завершается.

### 2. Переход на сайт продавца
**Основной поток событий:**
1. Вариант использования начинается, когда пользователь имеет список результатов поиска.
2. Пользователь нажимает на найденный товар.
3. Выполняется переход на сайт продавца.
4. Вариант использования завершается.

### 3. Загрузка фото профиля
**Основной поток событий:**
1. Вариант использования начинается после авторизации пользователя.
2. Пользователь нажимает на фото своего профиля.
3. Пользователю отображается окно для выбора фотографии с хранилища его компьютера.
4. Пользователь выбирает фотографию.
5. Если пользователь выбрал фотографию(документ) не формата img или же размер фото неудоволетворяет требованиям - 
в этом случае пользователь увидит ошибку на экране. Переход к п.7. Иначе переход к п.6
6. Фотография загружается в профиль и отображается пользователю.
7. Вариант использования завершается.

### 4. Добавление товара в избранное
**Основной поток событий:**
1. Вариант использования начинается после авторизации пользователя.
2. Пользователь нажимает на зведочку в правом верхнем углу отображенного товара.
3. Пользователю отображается окно уведомления о добавлении товара в избранное.
4. Звездочка меняет цвет.
5. Вариант использования завершается.

### 5. Просмотр избранных товаров
1. Вариант использования начинается после авторизации пользователя.
2. Пользователь нажимает на кнопку "My Favourites".
3. Пользователю отображается страница избранных товаров.
4. Вариант использования завершается.

### 6. Удаление товара из избранного
1. Вариант использования начинается после загрузки страницы "My Favourites".
2. Пользователь нажимает на красный крест в правом верхнем углу товара.
3. Пользователю отображается модальное окно подтверждения удаления.
4. Пользователь нажимает "Отменить" - переход к п.6. Пользователь нажимает "Удалить" - переход к п.5
5. Пользователю отображается список товаров без удаленного.
6. Вариант использования завершается.
